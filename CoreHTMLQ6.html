Following are some list of tags used in HTML

  1. <a> for link
  2. <b> to make bold text
  3. <strong> for bold text with emphasys
  4. <body> main HTML part
  5. <br> for break
  6. <div> it is a division or part of an HTML document
  7. <h1> for titles
  8. <i> to make an italic text
  9. <img> for images in document
  10. <ol> is an ordered list
  11. <ul> for an unordered list
  12. <li> is a list item in bulleted (ordered list)
  13. <p> for paragraph
  14. <span> to style part of text
